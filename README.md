# About Poem For Art
This is an app for art and poem lovers to capture ideas and get inspired in their creation of Triplets from artwork and Gemini's association.
The app is developed on Google AI Studio.

# For Developers
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
